## Usage Instructions

1. **Basic Synthesis**: Use `synthesize_speech.py` with different voice IDs
2. **Voice Interpolation**: Use `synthesize_speech_enhanced.py` with interpolation parameters
3. **Voice Morphing**: Use the morphing features in the enhanced synthesizer
4. **Voice Continuum**: Explore the voice space with continuum generation
5. **Voice Family**: Generate related voice variations
6. **Voice Analysis**: Analyze voice characteristics and quality

## Next Steps

1. Install required dependencies: `pip install -r requirements.txt`
2. Run the setup script: `python setup_hybrid_system.py`
3. Train models with: `python train_models.py`
4. Generate real audio with: `python synthesize_speech_enhanced.py`
5. Explore voice features with the demonstration scripts
