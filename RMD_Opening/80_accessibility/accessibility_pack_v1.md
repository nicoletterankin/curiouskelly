# Accessibility Pack

This document provides the official accessibility guidelines for all related media, including video content and promotional images. Following these standards ensures our content is inclusive and accessible to all audiences.

---

## 1. Captions Style Guide

This guide outlines the standards for all on-screen captions to ensure clarity, readability, and consistency.

*   **Case:** All captions must use sentence case.
    *   *Example:* `This is the first step.` (Not `this is the first step.` or `This Is The First Step.`)

*   **Sound Effects (SFX):** All significant, non-speech sounds must be described in brackets.
    *   *Example:* `[hammer clangs]`
    *   *Example:* `[soft, ambient music begins]`

*   **Line Limit:** Captions must not exceed **two (2) lines** on screen at any time.

*   **Character Limit:** Each line must not exceed **42 characters**, including spaces.

*   **Combined Example:**

    ```
    [sparks sizzle]
    Now we can begin the real work.
    ```
