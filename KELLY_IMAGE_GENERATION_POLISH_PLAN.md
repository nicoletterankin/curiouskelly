## üîç Brand Alignment Checklist

### Character Consistency (Non-Negotiable)

#### Facial Features
- [ ] **Face Shape:** Oval face (not round, not square)
- [ ] **Skin Tone:** Warm light-medium (not pale, not dark, not cool-toned)
- [ ] **Complexion:** Clear, smooth, natural glow (not matte, not overly shiny)
- [ ] **Eyes:** Warm brown, almond-shaped (not blue, not round, not small)
- [ ] **Eyebrows:** Dark brown, well-defined, natural arch (not thin, not unibrow)
- [ ] **Eyelashes:** Long, dark (not short, not light)
- [ ] **Hair Color:** Medium brown with caramel/honey-blonde highlights (not pure brown, not blonde, not black)
- [ ] **Hair Texture:** Soft waves that fall together cohesively (NOT frizzy, NOT whispy, NOT curly)
- [ ] **Hair Cohesion:** Hair strands stay together in unified sections (no separate curls, no frizz, no flyaways)
- [ ] **Hair Finish:** Smooth polished texture (not frizzy, not unkempt)
- [ ] **Hair Parting:** Slightly off-center or down the middle (not side part)
- [ ] **Lips:** Full, natural rosy-pink (not thin, not bright red)
- [ ] **Smile:** Genuine, warm, shows straight white teeth (not closed mouth, not fake)
- [ ] **Smile Lines:** Natural nasolabial folds (not absent, not exaggerated)
- [ ] **Eye Crinkles:** Slight crinkles at outer corners when smiling (not absent, not too pronounced)

- [ ] **Age:** Late 20s to early 30s (not younger, not older)
- [ ] **Build:** Athletic, strong capable presence (not thin, not heavy)

## üìö Knowledge Base

### What Works

#### Character Consistency Prompts
- **Template:** "Maintain exact facial features, hair color (medium brown with caramel/honey-blonde highlights), skin tone (warm light-medium), eye color (warm brown almond-shaped), and overall appearance from reference images."
- **Effectiveness:** High - consistently produces accurate character
- **Usage:** Include in all Kelly prompts

#### Reference Image Usage
- **Best Practice:** Use wardrobe-specific reference images
- **Daily Lesson:** `kelly_profile.png` (Director's Chair variant)
- **Reinmaker:** `kelly_three_quarter.png` (Reinmaker outfit)
- **Effectiveness:** High - significantly improves consistency

#### Negative Prompts
- **Template:** Mandatory negative prompts block unwanted styles
- **Effectiveness:** High - prevents common mistakes
- **Usage:** Always include in all prompts

### What Doesn't Work

#### Common Failures
1. **Generic Prompts:** "A woman named Kelly" - too vague
2. **Missing Reference Context:** Not mentioning reference images
3. **Weak Negative Prompts:** Not blocking unwanted styles
4. **Inconsistent Wording:** Changing character description

#### Edge Cases That Fail
1. **Extreme Angles:** Top-down or bottom-up views
2. **Busy Backgrounds:** Complex scenes distract from Kelly
3. **Action Poses:** Running/jumping can distort features
4. **Different Lighting:** Dramatic lighting can change appearance
