## STEP 11: Add Hair (The Realistic Hair Challenge)

### The Hair Problem You Identified:

**You're RIGHT about the hair issue:**
- Most CC5/iClone hair = soft, flowy, game-like
- Realistic sitting/portrait hair = weighted, structured, settled
- Hair should "sit" not "float"

### Best Hair Solutions for Kelly:

**Option 1: CC5 HD Hair Pack (Recommended)**
1. **Access Content Manager:**
   - Top menu: **"Content"** 
   - Click **"Content Manager"** or **"Smart Gallery"**
   
2. **Navigate to Hair:**
   - Left panel: Find **"Hair"** category
   - **LEFT CLICK to expand**
   - Look for subcategory: **"Hair HD"** or **"Realistic Hair"**

3. **Look for these hair types (more realistic):**
   - "Long Straight" (sits naturally)
   - "Professional" or "Business" styles
   - "Ponytail" (weighted, pulled back)
   - **Avoid:** "Wavy", "Flowing", "Dynamic" = too game-like

4. **Preview hair before applying:**
   - **LEFT CLICK on hair thumbnail**
   - Preview appears on character
   - If you like it: **Double LEFT CLICK** or click "Apply" button

**Option 2: Purchase Realistic Hair Pack**

**Best marketplace for realistic hair:**
- **ActorCore** (Reallusion's marketplace)
  - Website: actorcore.reallusion.com
  - Search: "HD Hair" or "Realistic Hair"
  - Filter: Female, Long, Straight
  
**Recommended packs:**
- "Ultra Realistic Hair Pack"
- "Professional Hair Collection"  
- "Portrait Hair Bundle"
- Look for: "physics-weighted" or "strand-based"

**How to install purchased content:**
1. Download from ActorCore
2. Files save to: `Documents\Reallusion\Content\`
3. In CC5: **Content ‚Üí Smart Gallery ‚Üí Refresh**
4. New hair appears in Hair category

**Option 3: Customize Existing Hair**

1. **Apply any long hair first** (even if too flowy)
2. **Access Hair Modification:**
   - Select character
   - **Modify ‚Üí Hair** section
   - Find **"Physics"** or **"Dynamics"** settings

3. **Reduce bounce/flow:**
   - "Stiffness": Increase to 80-100% (less movement)
   - "Damping": Increase to 70-90% (settles faster)
   - "Gravity": Increase (hair falls down naturally)
   - "Wind": Set to 0 (no artificial wind)

4. **Shape the hair:**
   - "Hair Length": Adjust
   - "Hair Volume": Reduce for tighter style
   - "Front Hairline": Adjust position
   - "Part Line": Set center or side part

**Option 4: Import Custom Hair (Advanced)**

If you have:
- Blender hair model
- FBX hair file
- OBJ hair mesh

You can import:
1. **File ‚Üí Import**
2. Select FBX or OBJ file
3. Position on head
4. Set as hair object

## STEP 13: Skin Material Refinement

**For photorealistic skin:**

1. **Select character** (LEFT CLICK on character in viewport)

2. **Access Material Editor:**
   - **Modify ‚Üí Material ‚Üí Skin** section
   OR
   - Right panel: Find "Material" icon (looks like sphere)

3. **Key skin settings:**

   **Roughness:** (how matte/shiny)
   - Value: 0.35-0.45 (Kelly's skin)
   - Slider or number input
   - Lower = shinier, Higher = more matte

   **SSS (Subsurface Scattering):** (skin translucency)
   - Value: 0.25-0.35
   - This gives realistic skin depth
   - Essential for photorealism

   **Normal Map Strength:** (pore detail)
   - Value: 0.8-1.2
   - Shows skin texture detail

   **Specular:** (highlights)
   - Value: 0.4-0.6
   - Controls shininess intensity

4. **T-Zone adjustment** (forehead/nose may be shinier):
   - Look for "T-Zone Roughness" 
   - Reduce slightly from base roughness
   - Or apply shine map if available

- Increase Physics Stiffness to 80-100%

### "Export to iClone fails"
