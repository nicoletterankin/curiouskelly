## üîç Quick Reference

### Common Commands

**Generate Kelly Assets:**
```bash
python -m kelly_pack.cli build --outdir ./output
```

**Run Digital Kelly:**
```bash
cd digital-kelly/apps/kelly_app_flutter
flutter pub get && flutter run
```

**Test Audio2Face:**
```bash
python kelly_audio2face/test_kelly_integration.py
```

### Key Directories

- `kelly_pack/` - Asset generation toolkit
- `digital-kelly/` - Flutter + Unity app
- `kelly_audio2face/` - Audio2Face integration
- `iLearnStudio/` - Creative pipeline
- `lesson-player/` - Web lesson player
- `synthetic_tts/` - TTS system
